**整体操作逻辑**
这个系统可以按“计划下达 -> 批次释放 -> 现场执行 -> 质量拦截 -> 维修解锁 -> 看板监控”的 MES 闭环来操作。

**1. 登录与角色选择**

```
账号：admin 密码：123456 
```

然后选择演示角色。不同角色影响页面按钮权限：

- 工厂管理层：偏查看全局数据、看板、驾驶舱。
- 生产主管：可新建工单、释放工单、暂停/关闭工单、强制出站、查看全局生产。
- 班组长：可查看负责产线，可执行部分批次、进站、上料、出站、维修协同操作。
- 操作工：主要执行进站、上料、出站、批次查看。
- 质量工程师：主要处理阈值配置、质量拦截、批次锁定/解锁审核。
- 维修员：主要处理维修接单、维修记录、维修提交。

**2. 首页生产驾驶舱**
登录后默认进入“首页 / 生产驾驶舱”。

这里用于管理层、生产主管、班组长快速判断现场状态：

- 顶部看当日工单总数、在制批次数、整体直通率、计划完成率。
- 中间看每条产线当前工单、产品型号、进度、OEE、设备状态。
- 右侧看异常告警，比如批次锁定、设备故障、物料缺料、质量异常。
- 点击产线卡片会进入产线状态看板。
- 点击告警条目会跳转到对应处理页面，比如质量拦截、设备监控、上料校验。

这一页的逻辑是“先看异常，再看产线，再看工单进度和质量趋势”。

**3. 工单管理流程**
路径：生产调度 -> 工单管理

生产主管的标准操作步骤：

1. 点击“新建工单”。
2. 选择产品型号。
3. 选择分配产线。
4. 输入计划数量。
5. 选择交货期。
6. 提交生成工单。
7. 在工单列表中找到“待释放”工单。
8. 点击“释放”。
9. 系统模拟校验 BOM、工艺路线、参数模板是否完整。

1. 校验通过后，工单释放到产线，并生成生产批次。
2. 生产过程中可以对“生产中”工单执行暂停。
3. 已完成工单可以执行关闭。
4. 点击工单号进入工单详情，查看 BOM、工艺路线、批次列表、工序进度。

班组长进入此页面时，主要是查看工单状态和详情，编辑类操作会被禁用或拦截。

**4. 批次管理流程**
路径：生产调度 -> 批次管理

这里是工单释放后的核心批次管控页面。

标准操作步骤：

1. 按批次号、工单号、产品型号、状态、产线筛选批次。
2. 查看批次状态卡片，判断待进站、生产中、暂停、维修中、已完成、已锁定数量。
3. 对异常批次执行“锁定”，需要填写原因。
4. 对已锁定批次执行“解锁”，系统会判断是否为自动锁定。
5. 如果是系统自动锁定，比如直通率低于阈值，则不能手动解锁，必须走质量拦截处理。
6. 对待料、设备故障场景，可以执行暂停。
7. 异常解除后执行恢复。
8. 点击批次号进入批次详情。

批次详情里可以查看：

- 批次基础信息。
- 工序流转时间轴。
- 上料记录。
- 质量检测记录。
- 工艺参数记录。
- 并可跳转进站、出站、维修操作页。

**5. 进站操作流程**
路径：生产执行 -> 进站操作

这是操作工或班组长的现场操作页面，适合扫码枪和平板。

标准操作步骤：

1. 扫描或选择批次号。
2. 系统展示批次产品型号、计划数量、当前工序、上一工序出站情况。
3. 系统进行工序合规校验。
4. 扫描或选择当前工序可用设备。
5. 查看上料校验状态。
6. 如果上料未完成，点击“上料操作”跳转到上料校验。
7. 输入或确认进站数量。
8. 确认操作人和备注。
9. 点击“提交进站”。

1. 校验全部通过后，系统记录进站时间、设备、上料信息、操作人。
2. 成功后跳转到出站操作页。

拦截逻辑：

- 未选择设备，不能进站。
- 批次锁定，不能进站。
- 上料未完成时会提示去补齐上料校验。

**6. 上料校验流程**
路径：生产执行 -> 上料校验

标准操作步骤：

1. 查看当前批次、产品型号、当前工序、对应设备。
2. 查看上料完成率。
3. 扫描物料条码。
4. 扫描站位码。
5. 点击“扫码校验”。
6. 系统判断站位是否存在。
7. 系统判断物料料号是否与 BOM 一致。
8. 校验通过后，该站位自动标记为“已完成”。
9. 校验失败时，弹窗提示原因，比如料号不匹配、站位不存在、替代料无权限。

1. 所有关键站位完成后，返回进站操作继续提交。

页面中的颜色逻辑：

- 已完成：绿色。
- 待完成：黄色。
- 失败：红色。

**7. 出站操作流程**
路径：生产执行 -> 出站操作

标准操作步骤：

1. 查看批次号、产品型号、当前工序、进站时间、对应设备。
2. 填写或确认完工数量。
3. 填写不良数量。
4. 填写报废数量。
5. 如果不良数量大于 0，选择不良处置方式：
   - 维修
   - 报废
   - 强制出站
6. 如果选择强制出站，只有生产主管角色允许提交。
7. 查看自动采集的质量检测数据，比如 SPI、AOI、批次良率。
8. 填写操作人和备注。
9. 点击“提交出站”。

1. 系统记录出站时间、完工、不良、报废、质量数据和操作人。
2. 如果有不良且选择维修，会流转到维修管理。

拦截逻辑：

- 批次被质量系统锁定时，禁止出站。
- 普通操作工不能强制出站。
- 锁定原因会直接显示给操作人员。

**8. 维修管理流程**
路径：生产执行 -> 维修管理

标准操作步骤：

1. 查看待维修批次、今日已维修、维修合格率、超期预警。
2. 在待维修批次列表选择一个批次。
3. 点击“维修接单”。
4. 填写维修方法。
5. 选择维修结果。
6. 填写维修数量、报废数量。
7. 填写不良原因分析。
8. 填写预防措施。
9. 点击“维修提交”。

1. 如果维修完成后需要解锁，由质量工程师点击“审核解锁”。
2. 审核通过后，批次恢复生产流转。

**9. 批次跟踪流程**
路径：生产执行 -> 批次跟踪

这个页面用于追溯。

标准操作步骤：

1. 扫描或输入批次号。
2. 点击查询。
3. 查看完整工序时间轴。
4. 查看每道工序的进站、出站、设备、操作人、完工数量、不良数量。
5. 查看上料记录汇总。
6. 查看操作日志。
7. 点击“导出追溯报告”生成追溯数据。

适用场景：

- 客诉追溯。
- 质量复盘。
- 批次异常定位。
- 工艺参数与质量数据核对。

**10. 设备监控流程**
路径：生产执行 -> 设备监控

标准操作步骤：

1. 查看设备总数、运行、待机、故障、离线数量。
2. 按设备类型筛选，比如印刷机、SPI、贴片机、回流炉、AOI。
3. 按设备状态筛选，比如运行、待机、故障、离线。
4. 按产线筛选。
5. 切换列表视图或矩阵视图。
6. 故障设备会红色高亮。
7. OEE 低于目标值的设备会黄色预警。
8. 点击“故障提报”提交故障工单。
9. 点击“保养确认”更新设备保养状态。

1. 下方查看故障设备详情和保养提醒。

**11. 质量阈值配置流程**
路径：质量管理 -> 阈值配置

质量工程师或生产主管操作：

1. 点击“新增配置”。
2. 选择产品型号。
3. 选择工序类型，比如 SPI 检测或 AOI 检测。
4. 输入检测项目。
5. 设置阈值下限、阈值上限、允许偏差。
6. 选择处置动作：
   - 预警
   - 自动锁定批次
   - 禁止出站
7. 点击“提交生效”。
8. 系统模拟同步到生产执行环节。
9. 列表中可以启用/停用规则。

1. 点击“记录”查看修改历史。

**12. 质量拦截管理流程**
路径：质量管理 -> 拦截管理

这是质量异常闭环核心页面。

标准操作步骤：

1. 查看今日拦截批次、待处置、已处置、主因分布。
2. 在拦截批次列表选择待处置批次。
3. 查看触发规则，比如 SPI 直通率低于 95%。
4. 查看检测原始数据。
5. 查看不良项明细。
6. 点击“处置接单”。
7. 选择处置方式：
   - 维修返工
   - 批次解锁
   - 强制放行
   - 批次报废
8. 填写处置意见。
9. 填写原因说明。

1. 点击“处置提交”。
2. 如果需要解锁，质量工程师点击“解锁审批”。
3. 审批通过后，批次恢复生产流转。

注意：系统自动锁定的批次不能在批次管理页直接手动解锁，必须从质量拦截管理完成审批。

**13. 看板中心操作**
路径：

- 看板中心 -> 产线状态看板
- 看板中心 -> 质量监控看板
- 看板中心 -> 生产进度看板

产线状态看板用于车间大屏：

1. 查看总产线数量、运行产线、整体稼动率、当日总产量、整体直通率。
2. 查看每条产线当前工单、产品型号、计划/完工、预计完成时间。
3. 查看设备状态色块。
4. 查看异常告警滚动区。
5. 查看当日工单进度排行。

质量监控看板用于质量部大屏：

1. 查看 SPI 直通率。
2. 查看 AOI 直通率。
3. 查看批次良率。
4. 查看不良率。
5. 查看拦截批次数。
6. 查看直通率趋势、不良类型分布、产线质量排行、拦截明细。

生产进度看板用于生产主管和管理层：

1. 查看 7 天内交货工单。
2. 查看未完成和延期预警。
3. 查看计划与实际对比甘特图。
4. 查看工单完成率明细。
5. 查看产线当日/月度产能达成率。

**14. 个人中心与消息中心**
个人中心：

1. 查看账号、姓名、部门、岗位、角色。
2. 修改密码。
3. 设置默认首页。
4. 设置列表每页条数。
5. 设置刷新频率。
6. 查看个人操作日志。

消息中心：

1. 按全部、告警通知、生产通知、质量通知、系统通知筛选。
2. 未读消息有角标。
3. 告警类消息红色置顶感更强。
4. 点击“详情”查看完整内容。
5. 点击“跳转”进入关联业务页面。
6. 点击“全部已读”批量处理。
7. 点击“删除”移除消息。

**推荐实际使用顺序**
日常生产最顺的操作路径是：

1. 生产主管登录。
2. 首页查看全局状态。
3. 进入工单管理，新建并释放工单。
4. 进入批次管理，确认批次生成和状态。
5. 操作工登录。
6. 进入上料校验，完成站位物料校验。
7. 进入进站操作，扫码批次和设备，提交进站。
8. 进入出站操作，确认产量和质量数据，提交出站。
9. 若有不良，进入维修管理处理。

1. 若质量阈值触发，质量工程师进入拦截管理处理和解锁。
2. 班组长或主管通过设备监控和批次跟踪定位现场问题。
3. 管理层通过看板中心观察产线、质量、进度。
